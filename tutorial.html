节点使用全指南

1. 获取你的“钥匙”

第一步：加入组织 通过主页按钮加入 QQ 群（如跳转失败，请手动搜群号或在浏览器打开）。
第二步：复制订阅 在【群公告】中寻找包含 http:// 关键字的长链接。请务必“长按全选”并完整复制，它是你连接世界的钥匙。

2. 准备“交通工具”

iOS 用户 推荐使用 Shadowrocket (需海外 ID 下载)。
Android / PC 用户 推荐使用 Clash Meta (功能最稳) 或 v2rayNG (操作最简)。具体包请在群文件下载。

3. 开启连接

导入订阅 打开软件 -> 配置/配置 -> 点击“+”或“从 URL 导入” -> 粘贴第一步复制的链接。
选择节点 点击“保存”或“更新订阅”，在节点列表中选择一个延迟较低（绿色数字）的节点，拨动顶部开关。
